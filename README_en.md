# Thunderbolt Core

Thunderbolt Core is the shared AE2 optimization and infrastructure layer for
AE2 Lightning Tech.

- Mod id: `thunderbolt`
- Java namespace: `com.moakiee.thunderbolt`
- Maven coordinate: `com.moakiee.thunderbolt:thunderbolt:2.0.0`

Build:

```powershell
.\gradlew.bat build
```

Publish to the local Maven repository for AE2LT development:

```powershell
.\gradlew.bat publishToMavenLocal
```
