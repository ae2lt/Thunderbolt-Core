# Thunderbolt Core 中文文档

Thunderbolt Core 是 AE2 Lightning Tech 使用的 AE2 优化与底层基础设施前置模组。

- 模组名：`Thunderbolt Core`
- Mod ID：`thunderbolt`
- Java 命名空间：`com.moakiee.thunderbolt`
- Maven 坐标：`com.moakiee.thunderbolt:thunderbolt:2.0.0-alpha.1`

构建：

```powershell
.\gradlew.bat build
```

发布到本地 Maven，供 AE2LT 开发环境依赖：

```powershell
.\gradlew.bat publishToMavenLocal
```
